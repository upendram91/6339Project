Observation

The model recommends the unique businesses as recommendations based on the previous experiences, reviews and check-ins information. Once the user data is analyzed all the top recommendations are sent back as output. The project has some functions defined especially to evaluate the accuracy of the model for a particular user whose previous experiences are recorded in the yelp dataset. For this model the accuracy was measured by considering 80% of the yelp data review.json as train data and the remaining 20 % data is considered as test set and accuracy is recorded as 75%. The current proposed model works best when the users have data either in the form of reviews or check-ins. For those who do not have either of them, we consider only the friends network and only the recommendations from the foodies are considered and returned as recommendations.
